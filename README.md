# Ahara

Ahara is a nutrition intelligence platform that helps users build balanced meals visually and track calories, protein, carbohydrates, fats, and overall nutrition.

## Features

- Authentication
- Personalized nutrition profile
- Visual meal builder
- Macro tracking
- Meal history
- Nutrition insights

## Tech Stack

- React
- Vite
- Tailwind CSS
- Node.js (planned)
- Express.js (planned)
- MongoDB (planned)

## Status

Currently under development.
UI is ready inspiration from Stich AI and data is managed by myself.

## Target user

Gym freak , Fitness enthusiastic , Diet conscious , A person who want to either loose weight or gain , A person who want to stay helathy and track there Diet
